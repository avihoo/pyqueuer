PyQueuer

A convenience tool to automatically produce/consume message to/from Message Queue.

Usage:

  1. Register your account and login.
  2. Configure your MQ, data_store(path of file messages) in setting.
  3. Setup your plugin stacks for convenience access plugins (optional)
  4. Open consume page to listen to a queue/topic.
  5. Open send page to send a message either in file or data

Thanks for using. Enjoy.
